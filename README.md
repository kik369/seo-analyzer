# Free SEO Analyzer Tool

A lightweight, privacy-focused SEO analysis tool that runs entirely client-side.

## Features
- Website SEO score (0-100)
- Meta tag analysis (title, description, og tags)
- Heading structure check
- Image alt text validation
- Page load performance hints
- Mobile-friendliness indicators
- **No data stored or transmitted** — analysis happens in the browser

## Deploy to GitHub Pages
1. Create a new GitHub repo
2. Push this code
3. Enable GitHub Pages in Settings > Pages
4. Your tool is live at `https://<username>.github.io/<repo>/`

## Monetization
- Add affiliate links for SEO tools/books
- Add a "Premium Analysis" CTA linking to your services
- Use for lead generation for SEO consulting
